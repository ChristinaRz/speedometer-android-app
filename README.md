# Speedometer Android App

## Description
Android application that tracks real-time speed using GPS and alerts the user when exceeding a predefined speed limit.

## Features
 Real-time GPS speed tracking
 Custom speed limit setting
 SQLite database to store speed history
 Voice command support
 Text-to-Speech speed warnings

## Technologies
Java
Android Studio
SQLite
Location Services API
Speech Recognition API
Text-to-Speech API
